This project is a simple **Bingo game** written in JavaScript
It was designed as a learning project to practice **JavaScript, CSS, HTML, arrays, and basic game logic**.

---

## Overview

The game generates a **Bingo board** with different activites to do on your next big trip with family or friends, and lets players:

- Check off items as they are called
- Track progress towards completing a row, column, or diagonal
- Play multiple rounds

---

## Features

- Activity Bingo boards
- Check for winning conditions automatically
- Simple text-based interface for beginners
